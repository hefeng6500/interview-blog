326.3 的幂 #

给定一个整数，写一个函数来判断它是否是 3 的幂次方。如果是，返回 true ；否则，返回 false 。

整数 n 是 3 的幂次方需满足：存在整数 x 使得 n == 3x

示例 1：


输入：n = 27
输出：true

示例 2：


输入：n = 0
输出：false

示例 3：


输入：n = 9
输出：true

示例 4：

输入：n = 45
输出：false

提示：

-231 <= n <= 231 - 1

进阶：你能不使用循环或者递归来完成本题吗？

循环不断除以 3

/**
 * @param {number} n
 * @return {boolean}
 */
var isPowerOfThree = function (n) {
  if (n > 1) {
    while (n % 3 === 0) {
      n = n / 3;
    }
  }
  return n === 1;
};

数学方式

/**
 * @param {number} n
 * @return {boolean}
 */
var isPowerOfThree = function (n) {
  return (Math.log(n) / Math.log(3)).toFixed(10) % 1 === 0;
};
